# data/ ディレクトリのデータ概要

このディレクトリには、スクリプトの実行に必要な各種データを配置する。

## 同梱データ

### `SGY_fanqie.csv`
- **作成者**: 池田 証寿 (北海道大学名誉教授)
- **ライセンス**: CC BY-SA 4.0 (Creative Commons Attribution-ShareAlike 4.0 International)
- **内容**: 廣韻 3,874 小韻の反切代表字（首字）について音韻情報を記録した独自整備ファイル。
- **規模**: 3,964 行、20 列
- **特徴**: `sbgy.xml` や `廣韻.csv` とは独立した声母分類体系（三十六字母体系の宋代分類）を持ち、校勘注によって切韻系諸資料との比較が可能。

#### 主なカラム定義
- `hanzi` / `hanzi_alt`: 漢字（本字・別字）
- `word_id`: `sbgy.xml` の親字 ID（`w` を除いた 8 桁）
- `shengmu`: 声母（照三・審三・穿三 等）
- `qingzhuo`: 清濁（全清・次清・全濁・次濁）
- `she_kaihe_deng`: 攝・開合・等を一括記述（例: `通中1`・`山開3`）
- `fanqie_note`: 切韻・王一・王二・王三・唐韻・廣韻との異同を記録した校勘注
- `rhyme_id`: 韻 ID（例: `01東01`）
